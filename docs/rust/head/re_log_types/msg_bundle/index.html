Structs and functions used for framing and de-framing a Rerun log message in Arrow.

An example main message (outer) schema:

+---------------------------------------------+-----------------------------------------------------+
| timelines                                   | components                                          |
+---------------------------------------------+-----------------------------------------------------+
| [{timeline: log_time, type: 0, time: 1234}] | {rect: [{x: 0, y: 0, w: 0, h: 0}], color_rgba: [0]} |
+---------------------------------------------+-----------------------------------------------------+

The outer schema has precisely 2 columns: timelines, components (TODO(john) do we want to add MsgId?)

The timelines schema is fixed and is defined by the [ArrowField] implementation on TimePoint.

The components schema is semi-flexible: it should be a [StructArray] with one column per component. Each component schema is defined in crate::component_types.
